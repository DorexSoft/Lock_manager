# Lock Manager 0.1

Простой консольный менеджер для хранения паролей и их генерации.

## Возможности
- 🗝️ Генерация числовых паролей
- 💾 Сохранение паролей в txt файлы
- 📖 Просмотр ранее сохраненных данных

## Планы развития
- Шифрование данных
- Поиск по сервисам
- Удаление и редактирование записей

## Технологии
- C++
- Стандартная библиотека (iostream, fstream)
