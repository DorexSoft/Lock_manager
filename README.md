# Lock Manager 0.3

Простой консольный менеджер для хранения паролей и их генерации.

## Возможности
- 🗝️ Генерация надежного пароля
- 💾 Сохранение паролей в зашифрованном виде
- 📖 Просмотр ранее сохраненных данных в обычном виде
-  🗑 Очистка всех ключей
- 📖 Функция с описанием и решением ошибок

## Технологии
- C++
- Стандартная библиотека (iostream, fstream)
